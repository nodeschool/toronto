Nodeschool Toronto

About

NodeSchool is a selection of open source workshops that teach web software skills. You can do them on your own or at one of the monthly NodeSchool Toronto events.

Community

The NodeSchool community is open to anyone. Whether you've been able to attend an event or not, you can join the discussion around learning Node in the Slack team.

Events

NodeSchool Toronto events are run by an enthusiastic group of volunteers. The workshops will be held monthly and will always be free.

The events offer a low-key environment to learn or practice Node and are generally three hours long.

Next event: June 20th at Shopify

Setup

Our recommended way to install Node, if you haven't already, is by using Homebrew. First, visit brew.sh and install Homebrew by following the instructions. Next, install Node by running the command brew install node to get the latest version.

Now, to get started with the first workshop, run the following in your terminal:

$ npm install -g learnyounode
$ learnyounode
